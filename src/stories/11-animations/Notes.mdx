# Animations

**Built-in:** `animate-spin`, `animate-ping`, `animate-pulse`, `animate-bounce`

**Custom animations:** Define in `tailwind.config.js` under `theme.extend.keyframes` and `theme.extend.animation`

**Example custom:**
```javascript
keyframes: {
  fadeIn: {
    '0%': { opacity: '0' },
    '100%': { opacity: '1' },
  }
}
animation: {
  'fade-in': 'fadeIn 0.5s ease-in-out',
}
```

**Usage:** `animate-spin`, `animate-pulse`, `animate-bounce`, `animate-ping`
