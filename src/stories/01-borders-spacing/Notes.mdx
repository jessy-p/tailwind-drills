# Borders & Spacing

**Padding:** `p-4`, `px-4`, `py-2`, `pt-4`, `pr-4`, `pb-4`, `pl-4`

**Margin:** `m-4`, `mx-4`, `my-2`, `mt-4`, `mr-4`, `mb-4`, `ml-4`

**Rounded:** `rounded-sm`, `rounded`, `rounded-md`, `rounded-lg`, `rounded-xl`, `rounded-full`

**Shadow:** `shadow-sm`, `shadow`, `shadow-md`, `shadow-lg`, `shadow-xl`

**Borders:** `border`, `border-2`, `border-4`, `border-gray-300`

**Hover:** `hover:bg-blue-600`, `hover:shadow-lg`, `hover:border-blue-500`

**Transition:** `transition`, `transition-all`, `duration-200`, `duration-300`, `ease-in-out`

**Cursor:** `cursor-pointer`

**Space Between:** `space-y-4`, `space-x-4`

**Divide:** `divide-y`, `divide-gray-200`

## Key Learnings

**Borders:** Always need both `border` (for 1px width) + color class like `border-gray-300`. Thicker borders use `border-2`, `border-4`, etc.

**Image Aspect Ratio:**
- `object-cover` — Fills container, maintains aspect ratio, crops if needed
- `object-contain` — Fits entire image, maintains aspect ratio, may have empty space
- `object-fill` — Stretches to fill (distorts aspect ratio)
- `object-scale-down` — Uses original size if smaller, scales down otherwise
- `aspect-{ratio}` classes — Set container ratio like `aspect-square`, `aspect-video`, `aspect-auto`
