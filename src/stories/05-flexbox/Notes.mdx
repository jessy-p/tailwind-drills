# Flexbox

**Container:** `flex`, `inline-flex`

**Direction:** `flex-row`, `flex-row-reverse`, `flex-col`, `flex-col-reverse`

**Wrap:** `flex-wrap`, `flex-wrap-reverse`, `flex-nowrap`

**Justify (main axis):** `justify-start`, `justify-end`, `justify-center`, `justify-between`, `justify-around`, `justify-evenly`

**Align items (cross axis):** `items-start`, `items-end`, `items-center`, `items-baseline`, `items-stretch`

**Align self:** `self-start`, `self-end`, `self-center`, `self-stretch`

**Gap:** `gap-4`, `gap-x-4`, `gap-y-2`

**Flex sizing:** `flex-1`, `flex-auto`, `flex-initial`, `flex-none`

**Grow/Shrink:** `grow`, `grow-0`, `shrink`, `shrink-0`
