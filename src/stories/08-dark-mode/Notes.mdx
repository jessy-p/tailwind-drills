# Dark Mode

**Enable:** Add `class="dark"` to html or parent element

**Variant:** `dark:bg-gray-800`, `dark:text-white`, `dark:border-gray-700`

**Common patterns:**
- Light bg: `bg-white dark:bg-gray-900`
- Light text: `text-gray-900 dark:text-white`
- Borders: `border-gray-300 dark:border-gray-700`
- Hover in dark: `dark:hover:bg-gray-800`
