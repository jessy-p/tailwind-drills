# Container Queries

**Container type:** `@container`, `@container/name`

**Query syntax:** `@sm:`, `@md:`, `@lg:`, `@xl:`

**Common use:** Responsive components that adapt to container size, not viewport

**Example pattern:**
```
Container: `@container`
Child: `@md:grid-cols-2` (changes at container's md breakpoint)
```

**Named containers:** `@container/sidebar`, then use `@md/sidebar:`
