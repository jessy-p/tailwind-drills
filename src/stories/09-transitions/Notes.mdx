# Transitions

**Base:** `transition`, `transition-all`, `transition-colors`, `transition-opacity`, `transition-transform`

**Duration:** `duration-75`, `duration-100`, `duration-150`, `duration-200`, `duration-300`, `duration-500`, `duration-700`, `duration-1000`

**Timing:** `ease-linear`, `ease-in`, `ease-out`, `ease-in-out`

**Delay:** `delay-75`, `delay-100`, `delay-150`, `delay-200`, `delay-300`

**Common patterns:**
- Button: `transition-colors duration-150`
- Modal: `transition-opacity duration-300`
- Slide: `transition-transform duration-200`
